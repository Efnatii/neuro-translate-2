# 0.1.1 - Release/Packaging Hardening

- Added repeatable packaging scripts:
  - `build:clean`
  - `build:zip`
  - `lint:manifest`
  - `smoke:edge`
- Added `tools/manifest-audit.js` with CI-friendly JSON report and non-zero exit on critical findings.
- Added `tools/build-zip.js` and `tools/build-clean.js` for release ZIP generation.
- Added `extension/buildInfo.json` generation (version, git SHA, build time).
- Added release playbook in `docs/RELEASE.md`.
- Added unit coverage for manifest audit bad-case detection.
